TritonAI Developer APIs

A governed gateway to self-hosted and approved cloud models, with starter credits, usage reporting, and recharge paths for projects that scale.

Access Request

Submit a request via the Kuali Build form (search "TritonAI API" in the Service Portal). Default credits ($15/month) are active for experimentation.

What You Get

  • API key for the LiteLLM gateway at tritonai-api.ucsd.edu/v1
  • Default credits ($15/month) for experimentation
  • Access to 20+ models — self-hosted (SDSC) and approved cloud (OpenAI, Anthropic, Google, Cohere)
  • Usage reporting and recharge mechanics for production projects
  • MCP-based connectors and reusable skills from the UCSD Skills Library

Model Hub Highlights

GPT-5.5, Claude Opus 4.7, Claude Sonnet 4.6, Gemini 3.1 Pro/Flash, Amazon Nova Premier, Cohere transcription, and self-hosted open-weight models inside the UC firewall.

Architecture

Your application → LiteLLM Gateway → On-prem models (SDSC) + Approved cloud models → Tools/Skills via MCP
