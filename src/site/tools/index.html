AI Tools at UC San Diego

These are not generic chatbots. They are focused tools built around specific service, instructional, research, and administrative workflows where grounded context and institutional trust matter.

Administrative Assistants

  • UC San Diego Assistant — campus knowledge from 70+ curated sites + ServiceNow KB
  • Job Description Helper — 83% faster drafting (30 min to 5 min)
  • Contract Reviewer (In Production) — 91% faster NDA/T&C review (120 min to 11 min). Automated monitored inbox + portal workflow for Procurement.
  • Fund Manager Coach — grant compliance and spending restriction guidance
  • Performance Review Evaluator — rating/comment alignment assessment
  • Project Charter Assistant — professional project charter drafting
  • Supplier Search Tool — campus supplier discovery and purchasing process
  • UCSD Benefit Assistant — employee benefit package questions
  • AI Workplace Opportunity Assistant (v0.1) — interviews users about daily tasks to suggest AI use cases

Student-Facing Tools

  • Class Scheduling Assistant (In Production) — AI-powered course scheduling launched July 2026, serving 20K-40K students in waves
  • Socratic Tutor — guided learning without direct answers

Academic & Research

  • Deep Research Assistant — synthesis across hundreds of research documents
  • Data Dictionary Assistant — definitions from Activity Hub and Blink
  • Instructor AI Assistant — auto-generates quizzes, summaries, and study guides from lecture content
  • Expert Notetaker — converts transcripts into structured documentation
  • Transcript Matching (In Production) — AI-powered OCR and pattern matching for incoming transcript-to-course equivalency verification

Security & Infrastructure

  • Phishing Analyzer — email risk scoring with warning sign explanations
  • Internet Search Assistant — conversational AI with real-time web access
  • General AI Assistant — writing and editing support without UC-specific context
  • Triton Developer Assistant — on-demand coding support and documentation analysis

Platform Tools

  • Enterprise Data Agent — natural language to structured enterprise data warehouse queries
  • TritonAI Harness — campus agentic development platform (V1 complete June 2026)
  • UCSD Skills Library — reusable MCP-based skills for the TritonAI ecosystem
