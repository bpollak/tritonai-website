Change Log

Changes to TritonGPT are documented below — they are searchable, listed chronologically, and can be sorted by each column.

TritonGPT Updates
Year Date What We Updated Description
2026 July 19 AI Contract Reviewer and Transcript Matcher in production

Two new TritonAI-powered tools are now in production. The AI Contract Reviewer monitors a dedicated inbox and portal, cleans formatting, extracts clauses, applies UC San Diego rulesets, and produces annotated redlines for NDAs and vendor agreements. The Enrollment Management Transcript Matcher processes incoming student transcripts, matching them to UC San Diego course equivalencies with 99.86% accuracy (3,700+ transcripts processed, 5 errors). Both tools leverage the TritonAI platform's on-prem inference and enterprise data integrations.
2026 July 9 Student Scheduling Assistant launched

The Student Scheduling Assistant is now live, serving 20,000-40,000 students in waves. Powered by TritonAI, the assistant leverages natural language scheduling interactions to help students plan and register for classes. Class Planner go-live scheduled for July 20.
2026 June 26 TritonAI Harness V1 feature set complete

The TritonAI Harness — UC San Diego's campus agentic development platform — reached V1 feature set completion. The Harness is the execution layer that turns model inference into governed, tool-using AI workers. It connects to the UCSD Skills Library, Onyx (417+ API endpoints), and LiteLLM (20K+ queries/week), with a companion Triton Code Agent daemon for cron scheduling, heartbeat monitoring, and automated job execution.
2026 June 17 Triton Code Agent Phase 1 and new models

Triton Code Agent Phase 1 built — a companion daemon for the T3 Code harness with cron scheduling, heartbeat monitoring, and job execution (9 source modules, 21 passing tests). New models added to the platform include GPT-5.5, Claude Opus 4.7, Claude Sonnet 5, Gemini 3.5 Flash, Mistral Large 3, Kimi K2.5, Nova Premier, and MiniMax M2. Self-hosted open-weight models (DeepSeek V4 Flash Max, GPT-OSS-120B, Llama, Mistral, Gemma 4 26b) are included in the $15/month free credits tier.
2026 June 16 Onyx contract extended and platform hardening

Onyx platform contract extended one year at flat rate, transitioning from startup vendor to strategic platform partner. The ucsd-memory MCP server (v1.1.0) was hardened with graceful session handoff for SSE transport and bearer token authentication. Onyx Canvas integration bugs are being addressed through an active crash program.
2026

May 27

UC San Diego Assistant launches in the UC San Diego mobile app

The UC San Diego Assistant is now available in the official UC San Diego mobile app on iOS and Android. Students, faculty, and staff can ask questions about campus resources, parking availability, dining options, library information, and more — directly from their phone. Powered by TritonGPT, the assistant delivers the same AI-powered campus knowledge available on Blink and the TritonGPT web interface, now in a mobile-friendly experience. 
2026

May 28

Platform update: Onyx 3.3

Upgraded to Onyx v3.3, bringing significant improvements to the platform that powers the UC San Diego Assistant's document search and retrieval capabilities. Behind-the-scenes hardening and observability improvements were also made to improve system performance and reliability. 
2026

May 1

TGPT Widget updates and live campus data

The TGPT widget has been updated to provide live parking availability and current busyness levels at Geisel Library. The interface has also been redesigned to make it easier to follow how the AI generates responses. Behind-the-scenes updates were also made to improve system performance and reliability. 
2026

April 14

New LLMs, UCSD Assistant tools, and search improvements

Adds GPT 5.4, Gemini 3.1 Pro/Flash, Gemma 4 26b, and Claude Sonnet 4.6, and retires the OpenAI 4 Series. The UC San Diego Assistant gains Parking and Busyness tools, the Internet Search Assistant moves to a more reliable web search, and answer quality and safety have been improved across the platform. More details can be found in the 4/14/2026 release notes.
2025

November 24

UI overhaul, personalization, deep research, and new LLMs.

This release brings significant updates to TritonGPT, featuring a refreshed Chat UI, an enhancement that organizes Documents and Chat Folders into "Projects”, and the introduction of next-generation models like GPT-5.1 and Gemini 3 Pro. Additionally, this version introduces a powerful new Deep Research tool and enhanced personalization features to make the platform work better for you.  More details can be found here.
2025

October 22

New LLM available

OpenAI's latest flagship model, GPT-5 is now available to use for all AI Assistants. 

 
2025

October 13

TGPT powered search results

Blink search results now feature an AI overview powered by the TritonGPT UC San Diego Assistant. 
2025

October 3

New default LLM and new Assistants

GPT OSS, OpenAI's open source model is now powering TritonGPT by default. Models are now being served through a centralized API service that allow for better model management. The new Data Dictionary Assistant helps report developers navigate the wealth of data available in the Activity Hubs. 
2025 September 12 Backend improvements

Upgraded to an improved embedding model which further improves accuracy of reseponses. Additionally, we're now serving all models via LiteLLM, ensuring a more efficient and scalable experience.

Users now have the ability to change their default LLM. Click on your initial in the upper right corner and select "User Settings". Select from the available models in the Default Model drop down. 
2025 August 28 Expert Notetaker

Paste a meeting transcript and get a detailed summary with action items.
2025 July 7 AI Chatbot Widget

The UC San Diego Assistant is now available as a Chatbot Widget on Blink. Find out more about the AI Chatbot Widget. 
2025 June 16 TritonGPT now available to students

Students now have access to TritonGPT. More information is available on TritonLink. 
2025 June 13 OpenAI GPT 4.1

GPT-4.1 is now available for all AI Assistants. It supports text and image inputs and delivers faster, more cost-effective performance with enhanced coding, reasoning, and instruction-following capabilities. This release also includes bug fixes for the My Documents feature. 
2025 May 23 Default LLM

Llama 4 Scout is now the default model, replacing Llama 3.3. This upgrade delivers significantly improved output quality.
2025 May 9 My Documents Feature

Allows users to create folders and upload documents, making it easy to organize and combine context from multiple files to generate new, cohesive content.
2025 Apri 25 UC San Diego Assistant

The UC San Diego Assistant now intelligently evaluates prompts to determine when UC San Diego-specific context should be applied. This enhancement improves response accuracy in scenarios where the General AI Assistant would provide a better fit, reducing irrelevant context injection and streamlining the user experience.
2025 April 2 UI overhaul, multi-LLM support, enhanced context, and user customization.

TritonGPT introduces a redesigned sidebar with grouped chat history, assistant-specific threads, and a new “Explore Assistants” panel for enhanced discoverability. A streamlined assistant selection interface and prompt shortcuts improve usability and workflow. Users can now choose between multiple LLMs—GPT-4o and LLaMA 3.3 70B—based on task needs. Backend upgrades include a 128,000-token context window. The new User Settings panel enables theme toggling, auto-scroll, keyboard shortcuts, and full chat deletion. Legacy folders and dropdowns have been removed in favor of a cleaner, persistent navigation model. More details can be found here. 
2025 February 14 Platform updates and UX enhancements

Upgraded to the latest version of Onyx and improved the Related Questions feature. Users can now click on related questions to automatically populate them in the prompt window for a seamless experience.
2025 January 14 Platform updates

Upgraded to the latest version of Onyx (formerly Danswer), integrating chat and search functionalities into a unified experience. The update also includes an upgrade to Llama 3.3 and the expansion of indexed websites—now totaling 63—to enrich the UC San Diego Assistant's contextual knowledge and improve its ability to answer a broader range of questions.
2024 December 13 JD Helper Tool Overhaul, Internet Search Assistant, "Related Questions" feature

The JD Helper tool has been revamped to improve its functionality and reliability. Once you've developed your Position Overview, ask it to create a job posting, interview questions and other collatoral related to the hiring process. 

The Internet Search Assistant has been introduced, providing real-time informaton and answers.

A "Related Questions" feature has been added to the Fund Manager Coach, UC San Diego and Internet Assistants, providing thought-provoking topics and suggestions at the end of each response, tailored to the context of the original question.
2024 November 26 UX enhancements and system updates

Cleaner interface that allows for easier toggling between AI Assistants. 
2024 November 8 Added Email Phishing Analyzer and general platform improvements

Upload or paste emails, including key details like sender, subject, and content. This assistant will analyze them, provide a risk rating, and explain the ranking.
2024 August 13 Feature additions and updated LLM

TritonGPT updated with a refreshed UI design and the ability to save and organize chats. Users can now upload and interact with documents using an AI Assistant. Additionally, users can search across all documents and websites, switch between Assistants within a chat flow, and access a Prompt Gallery to reuse previously created prompts. Other new features include the ability to see the context behind an Assistant's response, share chats with others, and interact with specific sources within the chat flow. A new AI Assistant, Fund Manager Coach, is now broadly available. View more detailed release notes.
2024 May 14 Campus rollout complete

TritonGPT is now available to all employees that are part of campus and health sciences, totalling close to 37,000 faculty, staff and student staff. If you are part of those units and are having difficulty accessing TritonGPT, contact the Service Desk. 
2024 April 29 Large Language Model (LLM)

TritonGPT now uses the Llama 3 LLM to power the UC San Diego Assistant and the General AI Assistant. Llama 3 was trained on a much larger dataset (15T tokens) compared to Llama 2 (2.2T tokens), resulting in improved performance across various benchmarks. Llama 3 offers better reasoning abilities, code generation, and instruction-following capabilities compared to Llama 2.

As part of this enhancement, an upgrade was made to the Virtual Large Language Model (VLLM) environment. VLLM is a powerful and versatile open-source library that makes it easier and more efficient to use large language models (LLMs) for inference and serving. 
2024 April 15 Authentication

TritonGPT now uses campus Single Sign-on for authentication. This replaces the previous authentication method which was the Google Authenticator. This deployment also included some minor user interface improvements.
2024 March 28 JD Helper

Enhancements and bug fixes to JD Helper: updated wording in the JD Helper flow to clarify what is needed in order to produce better outcomes in retrieving job cards. Enhanced JD Helper system prompt to provide more robust output. 
2024 March 20 TritonGPT

Launched production candidate that will support use by employees of campus and health sciences. Danswer enterprise code base and larger embedding model. User experience improvements. 
2024 February Hardware

Added additional hardware into the environment in perparation for campus launch. 
2023 October TritonGPT

Initial proof of concept version deployed and made available to over 400 pilot users.
