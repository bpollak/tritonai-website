April 14, 2026 Release Details

Overview

This release expands the models you can choose from in TritonGPT, adds new campus-aware tools to the UC San Diego Assistant, and improves the quality of answers from the Internet Search Assistant. A new page on the TritonAI site helps you decide which model is the best fit for what you're trying to do. Behind the scenes, search and safety systems have been upgraded to make responses faster, more reliable, and better aligned with campus policy.

What's New and Changed

New and Updated Language Models

You can now select from a refreshed lineup of large language models (LLMs) when starting a chat. The newly available models are:

  • GPT 5.4 — OpenAI's newest flagship model. A strong general-purpose choice for writing, summarizing, analysis, and complex reasoning tasks.
  • Gemini 3.1 Pro — Google's latest high-capability model, well suited to long documents, research, and tasks that need careful, structured thinking.
  • Gemini 3.1 Flash — A faster, lighter version of Gemini 3.1 designed for quick answers, drafting, and everyday questions.
  • Claude Sonnet 4.6 — Anthropic's latest balanced model. A good fit for careful writing, editing, and nuanced conversations.
  • Gemma 4 26b — An efficient open model option for fast, lightweight tasks.

The OpenAI 4 Series models have been retired and are no longer available. If you previously relied on one of those models, GPT 5.4 is the recommended replacement.

Choosing the Right Model

A new section on the TritonAI Models page explains what each model is best at and when to pick one over another, so you can match the model to your task without needing to memorize technical details.

 Deep Research

  • Search Capabilities Return: Internet search functionality is back in TritonGPT, allowing for up-to-date information retrieval directly within the platform. Use this feature combined with Deep Research to develop rich documents and analysis that combines information from UC San Diego and public sources.
  • How to Access: Access these features by selecting the Internet Search Assistant located within the Agent Selection Panel. 

UC San Diego Assistant — New Campus Tools

  • Parking — Ask the UC San Diego Assistant about campus parking and it can now look up relevant information to help answer your question.
  • Busyness — The assistant can now report how busy different campus locations are, helping you plan visits to dining, libraries, and other spaces.

The assistant's default responses have been tuned for more consistent, on-brand answers.

Internet Search Assistant — Improved Web Search

The Internet Search Assistant now uses a new, built-in web search system that returns more reliable, more current results with fewer errors.

Better Answer Quality and Safety

  • Improved document search. The system that finds information inside your documents and knowledge sources has been upgraded for faster, more accurate results.
  • Smarter retrieval for grounded answers. The model that pulls relevant context into answers (used in fast and contextual retrieval) has been updated, improving answer quality across assistants.
  • Stronger, more flexible safety guardrails. TritonGPT's safety layer can now apply UC San Diego–specific rules more precisely, helping ensure responses stay appropriate for campus use.
